3. Описание файла tax.xml

Представляет собой файл формата XML с корневым элементом <tax> со множеством дочерних элементов <row>, которые в свою очередь содержат дочерние элементы <query> с параметрами запроса для выборки.

3.1. Элемент <row>

Каждый элемент <row> соответствует строке выходной таблицы с кодом NNNNNN, где NNNNNN соответствует коду строки из файла etalon.xml. 

<row code='190120'>
    <query>
        <rozd>3</rozd>
        <bd>0</bd>
        <rd>1901</rd>
        <pg></pg>
        <coef>0.334</coef>
    </query>
    <query>
        <rozd>3</rozd>
        <bd>0</bd>
        <rd>1905</rd>
        <pg></pg>
        <coef>0.334</coef>
    </query>
</row>

Элемент <row> может содержать неограниченное количество элементов <query>. Для каждой строки, определенной в <row> результат равен сумме результатов дочерних элементов <query>.

3.2. Элемент <query>

Содержит следующие элементы:

  • rozd – столбец Раздел из файла-отчёта казначейства. Может быть пустым, если необходима выборка из всех разделов;
  • bd – вид бюджета: 0 – государственный, 1 – местный;
  • rd – раздел бюджетной классификации, 4 цифры;
  • pg – параграф бюджетной классификации, 2 цифры. Может быть пустым, если необходима выборка по всем параграфам;
  • coef – Коэффициент. Представляет собой рациональное число с двумя или более разрядами, причём в качестве разделителя выступает точка (1.00, 0.334, -2.35). Может быть отрицательным, если необходимо обеспечить вычитание какой-либо суммы из общей суммы по строке.

4. Описание файла config.xml

Представляет собой файл формата XML с корневым элементом <config>, содержащим содержат дочерние элементы <path> и <treasury_files>.

4.1. Элемент <path>

В дочерних элементах <path-bank> и <path-out> содержит информацию о каталогах:

  • <path-bank> — в который пользователем помещаются казначейские файлы;
  • <path-out> — в который сохраняется выходной HTML-файл;
<path>
    <path-bank>d:\Autobnk\in</path-bank>
    <path-out>d:\Autobnk\out</path-out>
</path>

4.2. Элемент <treasury_files>

Во множестве дочерних элементов <file> содержит информацию об органах государственной казначейской службы.

Каждый элемент <file> содержит следующие дочерние элементы:

  • <treasury> – наименование органа государственной казначейской службы. Используется только для его визуальной идентификции, поэтому может отличаться от официального наименования;
  • <code> – код органа государствееной казначейской службы;
  • <raj> – код района инспекции.

<treasury_files>
    <file>
        <treasury>УДКСУ у Криворізькому районі</treasury>
        <code>097</code>
        <raj>18</raj>
    </file>
</treasury_files>

2. Описание файла etalon.xml

Представляет собой файл формата XML с корневым элементом <table> со множеством дочерних элементов <row> и содержит информацию о наименовании строк таблицы и порядке их размещения.

<table>
    <row>
        <code>140100<code>
        <name>ПДВ<name>
        <nompp>1<nompp>
    <row>
</table>

  • code – код строки. Обычно состоит из 6 цифр, первые 4 – код налога в соответствии с бюджетной классификацией, две последние – выбираются произвольно и должны быть уникальными в пределах одного платежа (пример — коды строк платы за землю 130501, 130502…);
  • name – название платежа, отображаемое в строке;
  • nompp – порядковый номер строки.

2. Описание файла etalon.yaml

Представляет собой файл формата YAML с элементами etalons и footer. Оба они содержат множество дочерних элементов row, содержащих информацию о кодах строк таблицы, наименовании и порядке их размещения.

etalons:
  - code: 140100
    name: 'ПДВ'
    row: 1

  • code – код строки. Обычно состоит из 6 цифр, первые 4 – код налога в соответствии с бюджетной классификацией, две последние – выбираются произвольно и должны быть уникальными в пределах одного платежа (пример — коды строк платы за землю 180105, 180106…);
  • name – название платежа, отображаемое в строке;
  • nompp – порядковый номер строки.
